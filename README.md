# No comments
## What does it do?
It blocks comment sections and related comment elements.

## Why did you create this?
Because comments: waste time, are often useless and are distracting.

## How can I use it?
By adding `https://raw.githubusercontent.com/hansjansen2/no-comments/refs/heads/main/filter.txt` as a custom filter to your program. 

## How is support?
Good for Brave Shields and uBlock Origin (manifest version 2) on Firefox. Other programs might work, but I don't test for them.
